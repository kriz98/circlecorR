# circlecorR: Circular Correlation Wheel Plots

Draws MNE-style circular "correlation wheel" plots natively in R. The
main entry point is
[`corr_wheel()`](https://kriz98.github.io/circlecorR/reference/corr_wheel.md);
[`compute_correlations()`](https://kriz98.github.io/circlecorR/reference/compute_correlations.md)
builds the r/p input from raw data.

# Synthetic gastric-symptom dataset

A fully simulated dataset (not patient data) mimicking the structure of
a gastric-symptom study: demographics, gastric metrics, symptom
sub-scales, and summary scores for 100 imaginary subjects, one row per
subject. Loadings are deliberately moderate and uneven, so only a
realistic subset of correlations is significant. Generated by
`data-raw/make_example_data.R`.

## Usage

``` r
gastro_symptoms
```

## Format

A data frame with 100 rows and 16 variables:

- Age, BMI:

  Demographics.

- Amplitude, Fed-Fasted AR, Frequency, GA-RI:

  Simulated gastric metrics.

- Nausea, Early satiety, Bloating, Upper GI pain, Lower GI pain,
  Heartburn:

  Symptom sub-scale scores.

- GCSI, PAGI-SYM, PAGI-QoL, EQ-5D:

  Summary symptom and quality-of-life scores.

## Details

Bundled with the package (`LazyData: true`), so it is available directly
as `gastro_symptoms` as soon as you
[`library(circlecorR)`](https://kriz98.github.io/circlecorR/) – no
[`data()`](https://rdrr.io/r/utils/data.html) call needed. Pass it
straight to
[`corr_wheel()`](https://kriz98.github.io/circlecorR/reference/corr_wheel.md);
see
[`vignette("circlecorR")`](https://kriz98.github.io/circlecorR/articles/circlecorR.md)
for a worked example.

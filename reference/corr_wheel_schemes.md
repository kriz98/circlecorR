# List built-in colour schemes

Returns the names of the colour schemes bundled with the package, for
use as the `scheme` argument of
[`corr_wheel()`](https://kriz98.github.io/circlecorR/reference/corr_wheel.md).

## Usage

``` r
corr_wheel_schemes()
```

## Value

A character vector of scheme names.

## See also

[`corr_wheel_scheme()`](https://kriz98.github.io/circlecorR/reference/corr_wheel_scheme.md),
[`corr_wheel()`](https://kriz98.github.io/circlecorR/reference/corr_wheel.md)

## Examples

``` r
corr_wheel_schemes()
#> [1] "default"    "colorblind" "mono_blue"  "vivid"     
```
